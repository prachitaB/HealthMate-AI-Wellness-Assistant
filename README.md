# HealthMate – AI Health & Wellness Assistant

## SDG Goal

SDG 3 – Good Health and Well-being

## Project Overview

HealthMate is an AI-powered chatbot developed using Botpress to promote healthy habits, wellness awareness, and positive lifestyle choices.

## Problem Statement

Many people lack awareness about nutrition, exercise, hydration, and mental wellness, resulting in unhealthy lifestyles and preventable health issues.

## Features

* Healthy Diet Guidance
* Exercise Tips
* Hydration Reminders
* Mental Wellness Support
* Healthy Lifestyle Suggestions
* Motivation

## Technology Stack

* Botpress
* Conversational AI
* NLP
* GitHub
